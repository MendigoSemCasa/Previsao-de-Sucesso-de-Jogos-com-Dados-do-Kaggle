# Previsao-de-Sucesso-de-Jogos-com-Dados-do-Kaggle
O objetivo deste projeto é prever se um jogo de videogame será bem-sucedido com base em informações históricas como: Ano de lançamento Plataforma Gênero Publicadora (Publisher) O sucesso é definido como vendas globais acima da mediana do dataset. Trata-se de um problema de classificação binária supervisionada.
